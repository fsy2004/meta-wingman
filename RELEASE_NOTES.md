# Meta Wingman 0.3.0

本地运行的 Meta 分析 / 系统评价桌面工具。选一个具体分析、填参数、出图出表,数据不出本机。

这一版从早先的网页版改成了原生 Windows 桌面程序,并把方法从十来个大类拆成 61 个可单独运行的具体分析。

## 本版包含

- 61 个具体分析,分 11 类:数据准备与换算、核心合并、异质性与调节、小研究与发表偏倚、稳健性与影响、复杂数据结构、网络 Meta、诊断准确性、序贯与效能、证据确信度、报告。每一项是一张具体的图或表,不是一个大而全的方法。
- 数据助手,不必先把数据改成固定格式:
  - 列映射 —— 用你自己的列名,把它们对应到该分析需要的角色即可(程序会先自动猜)。
  - 粘贴录入 —— 从 Excel 直接粘贴数据行。
  - 转换器 —— 2×2、均值±SD、相关、CI、p 值互算成效应量与标准误;中位数/四分位→均值±SD。
- 结果分「图」「表」两栏。图可另存 PNG 或矢量 PDF,表可一键复制,也可打开输出目录。
- 图统一走 Arial,按 Nature 制图规范输出矢量 PDF 与 PNG。
- 中英双语,右上角切换,首次按系统语言。原生 Windows 界面。
- 内存红绿灯按你的数据估算峰值占用。
- 全程本地,数据不出本机。安装依赖默认走清华镜像,应用从 Gitee / GitHub 下载。

## 环境要求

Windows,Python 3.9 以上,R 4.x。版本过低时会给出升级方式,不会自动改动你的 Python / R。

## 安装与启动

1. 下载 `install.bat`(Gitee 或 GitHub)。
2. 双击 `install.bat` —— 拉取应用并安装 Python / R 依赖。
3. 双击 `start.bat` —— 打开桌面窗口。

## 已知限制

- 仅支持 Windows。
- 需要自己装好 Python 与 R;程序不会替你改动系统里的解释器。
- 网络 Meta 的 CINeMA 用贡献矩阵近似;诊断的 HSROC 为初步实现;个别高级分析后续会继续完善。
- 结果图已达出版级 Arial 矢量,Nature 细节(线宽、去粗体表头等)仍在逐步打磨。
